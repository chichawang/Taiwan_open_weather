# Taiwan_open_weather_data

針對氣象相關公開資料進行讀取和視覺化產品(目前只有繪圖)

建議開發環境
1. WSL (Windows Subsystem for Linux) + anaconda
2. Google Colab

需要套件
1. python3.7 above
   1.1 json
   1.2 numpy
   1.3 pandas
   1.4 matplotlib
   1.5 cartopy
   1.6 shapefile
   1.7 proj
   1.8 request
2. wget or curl

基本技術 command line、python 基本語法
